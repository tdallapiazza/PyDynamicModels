Exercice 2: Mode automatique

Mettre en place un mode automatique dans lequel le chariot se déplace d'un bout à l'autre du rail avec une temporisation de 2 secondes à chaque extrémités.

Consignes
  • Créer un deuxième POU pour le mode automatique
  • Utiliser les boutons du commutateur de mode pour basculer du mode manuel au mode automatique
  • Modifier le mode manuel pour ne s'exécuter que si le sélecteur de mode est sur "manuel"
  • Utiliser les capteur de fin de course pour déclencher un timer de 2s
  • En fonction du capteur activé, commuter le sense de marche
  • Vérifiez le fonctionnement dans le simulateur
  • Vérifiez que le mode manuel est toujours pleinement fonctionnel
