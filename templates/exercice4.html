Exercice 4: Fonctionnement avancé

Augmenter la complexité du mode automatique en ajoutant un arrêt à la position centrale. Cet arrêt a une durée maximale de 3 secondes. Dans cet interval de temps, l'opérateur peut choisir avec les boutons de direction du mode manuel dans quelle direction va repartir le chariot. Le chariot repart dès l'action utilisateur sans attendre la fin de la temporisation.

Si aucun ordre n'est donné, le chariot reprend sa route dans la direction avant l'arrêt central après le temps d'arrêt de 3 secondes. Contrairement à l'exercice 3, il est souhaité ici que si le bouton d'arrêt est basculé durant les 3 secondes de pause, la temporisation soit également interrompue. Le temps restant est alors repris dès que le sélecteur repasse sur la position "marche".

Consignes
  • Interfacer le capteur centrale (arrêt et temporisation)
  • Programer la nouvelle logique
  • Vérifier le nouveau fonctionnement du mode automatique
  • Vérifier ce qui se passe si le mode automatique est interrompu (bascule sur arrêt) durant la phase d'attente
  • L'arrêt et le démarrage du mode automatique subsiste
  • Le fonctionnement du mode manuel est inchangé
